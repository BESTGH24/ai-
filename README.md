AI-puls
欢迎使用 AI+ Thinking Framework —— 一套专为AI助手设计的深度思考与推理方针。本框架旨在帮助AI在响应前进行系统化、多维度、高效率的思考，从而输出更专业、准确且贴合用户需求的回答。

项目简介
ai-puls定义了一套完整的思考流程，涵盖从理解用户意图到生成最终答案的每一个环节。它强调动态调整分析深度、并行推理、缓存复用以及自我验证，使AI能够灵活应对从简单查询到复杂专业问题的各种场景。

核心文件 ai-plus.txt 包含了所有指导原则和思考步骤，可直接作为AI系统的推理模块参考。

主要特点
全面性：要求AI在回答前重述问题、分析背景、识别隐含需求，确保信息不被曲解。

深度推理：分步推理、多角度分析、结论后验证，特别适用于专业领域。

动态智能：根据查询的复杂性、利害关系、时间敏感性等因素自动调整思考深度和风格。

高效并行：采用流水线式、连续批处理式思考，消除推理间隙，实现无等待响应。

仿生学思考：换位模拟用户需求，建立思考间的联系，并在任务完成后总结更深层要点。

自验证机制：回答前回顾质疑，测试结论准确性，保证输出质量。

文件说明
ai-plus.txt — 核心指南文件，详细描述了思考前的准备、初始问题分析、深度问题思考、仿生发现与思考、测试与验证等所有步骤。

README.md — 本文件，项目介绍与使用说明。
